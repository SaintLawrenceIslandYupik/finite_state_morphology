## Version

This is version 1.1 of the St. Lawrence Island Yupik finite-state morphological analyzer.

This version uses an updated standardized set of tags that matches those used in version 2.0.

This is version was referred to as the *Chapter 10 Chen and Schwartz (2018) Analyzer*, or *Ch10 C&S* in [Improved Finite-State Morphological Analysis for St. Lawrence Island Yupik Using Paradigm Function Morphology](https://www.aclweb.org/anthology/2020.lrec-1.326/) by Emily Chen, Hyunji Hayley Park, and Lane Schwartz (LREC 2020).
